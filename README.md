## Steps followed to create the repository and push it to GitHub
 ---- 
 1. Go to my profile > New Repository **(Green button)**
 2.  Open VS code
      *  Create a new folder
      *  Open Terminal and run git init
      *  git remote add origin [The repo's URL]
      *  git commit -m "My message"
      *  git push
